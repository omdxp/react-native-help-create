# Getting Started

- To install it globally in your system run:

<Tabs groupId="package-manager">
  <TabItem value="npm" label="npm" default>
    <CodeBlock>npm i react-native-help-create -g</CodeBlock>
  </TabItem>
  <TabItem value="yarn" label="yarn">
    <CodeBlock>yarn global add react-native-help-create</CodeBlock>
  </TabItem>
</Tabs>

Or

To install it as a dev dependency in your react native project run:

<Tabs groupId="package-manager">
  <TabItem value="npm" label="npm" default>
    <CodeBlock>npm i react-native-help-create --save-dev</CodeBlock>
  </TabItem>
  <TabItem value="yarn" label="yarn">
    <CodeBlock>yarn add react-native-help-create --dev</CodeBlock>
  </TabItem>
</Tabs>

If you want to use it without installing it (using npm cache) run:

```sh
npx react-native-help-create --help
```
